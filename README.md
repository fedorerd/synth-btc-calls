# Synth Bet Advisor

Finds mispriced BTC up/down markets on Polymarket by comparing [Synth](https://synth.com)'s probabilistic forecasts against market prices. Uses a probability term structure to pick which market to trade and distribution-aware Kelly sizing to determine how much to bet.

Live at [synth.fedoras.dev](https://synth.fedoras.dev). See [/about](https://synth.fedoras.dev/about) for the full methodology.
